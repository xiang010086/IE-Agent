# 动作标注指南

> **目标**：帮助理解如何标注产线作业动作，用于验证动作识别功能
> **标注目的**：确认动作识别规则的准确性，不是训练模型

---

## 一、动作标注是什么？

### 1.1 简单理解

**动作标注 = 给视频中的每个动作打标签**

就像给照片加标签一样：
- 照片标签："这是猫"、"这是狗"
- 动作标签："这是伸手"、"这是抓取"、"这是移动"

### 1.2 为什么需要标注？

```
目的：验证动作识别规则是否正确

流程：
┌─────────────────────────────────────────────────────┐
│  1. 你看视频，标注：这段是"伸手"，这段是"抓取"       │
│  2. 系统分析视频，输出：这段是"伸手"，这段是"抓取"   │
│  3. 对比两者是否一致                                 │
│  4. 如果不一致，调整系统的识别规则                   │
└─────────────────────────────────────────────────────┘
```

**重要**：我们用规则判断动作，不是AI学习，所以标注是用来验证规则的，不是用来训练模型。

## 二、五种核心动作的标注方法

### 2.1 Reach（R）伸手动作

**怎么识别伸手？**

看视频时问自己：
```
1. 手是不是从身体附近向外移动？
2. 手是不是伸向某个物体？
3. 手臂是不是伸直了？
```

**标注示例**：

| 场景 | 是否伸手 | 原因 |
|------|----------|------|
| 工人伸手拿桌上的螺丝刀 | ✅ 是伸手 | 手向外移动，伸向物体 |
| 工人手放在桌上不动 | ❌ 不是伸手 | 手没有移动 |
| 工人伸手后停在空中 | ❌ 伸手已结束 | 伸手动作在停住时结束 |

**标注步骤**：
```
1. 找到"手开始向外移动"的时刻 → 记为开始时间
2. 找到"手停住"或"开始抓取"的时刻 → 记为结束时间
3. 在表格中填入：动作类型=R，开始时间=0.5秒，结束时间=1.2秒
```

### 2.2 Grasp（G）抓取动作

**怎么识别抓取？**

看视频时问自己：
```
1. 手是不是刚完成伸手？
2. 手是不是停住了？
3. 手是不是握住/拿起了物体？
```

**标注示例**：

| 场景 | 是否抓取 | 原因 |
|------|----------|------|
| 工人伸手后握住螺丝刀 | ✅ 是抓取 | 伸手后握住物体 |
| 工人手已经握着螺丝刀移动 | ❌ 抓取已结束 | 已经进入移动阶段 |
| 工人伸手后没拿到东西 | ❌ 不算抓取 | 没有成功获取物体 |

**标注步骤**：
```
1. 找到"手刚停住"的时刻 → 记为开始时间
2. 找到"手开始移动（带着物体）"的时刻 → 记为结束时间
3. 抓取通常很短（0.2-0.5秒）
```

### 2.3 Move（M）移动动作

**怎么识别移动？**

看视频时问自己：
```
1. 手是不是带着物体在移动？
2. 手是不是从A位置移到B位置？
3. 移动方向是不是比较稳定（不是乱晃）？
```

**标注示例**：

| 场景 | 是否移动 | 原因 |
|------|----------|------|
| 工人拿着螺丝刀从桌上移到工件处 | ✅ 是移动 | 带着物体稳定移动 |
| 工人伸手过程中（没拿到物体） | ❌ 不是移动 | 这是伸手，不是移动 |
| 工人拿着物体原地晃动 | ❌ 不算移动 | 没有明确目的地 |

**标注步骤**：
```
1. 找到"手开始带着物体移动"的时刻 → 记为开始时间
2. 找到"手停住"或"开始放开"的时刻 → 记为结束时间
3. 移动通常是较长的动作（0.5-3秒）
```

### 2.4 Release（RL）放开动作

**怎么识别放开？**

看视频时问自己：
```
1. 手是不是刚完成移动？
2. 手是不是松开物体？
3. 手是不是从握着变为放开？
```

**标注示例**：

| 场景 | 是否放开 | 原因 |
|------|----------|------|
| 工人把螺丝刀放在桌上松手 | ✅ 是放开 | 手松开物体 |
| 工人放下螺丝刀后手收回 | ✅ 是放开 | 放开动作在松手时完成 |
| 工人拿着螺丝刀继续干活 | ❌ 没有放开 | 还没松手 |

**标注步骤**：
```
1. 找到"手开始松开物体"的时刻 → 记为开始时间
2. 找到"物体脱离手"的时刻 → 记为结束时间
3. 放开通常很短（0.1-0.3秒）
```

### 2.5 Wait（W）等待动作

**怎么识别等待？**

看视频时问自己：
```
1. 工人是不是全身停住不动？
2. 停住时间是不是超过1秒？
3. 停住时是不是在等什么（等物料、等机器）？
```

**标注示例**：

| 场景 | 是否等待 | 原因 |
|------|----------|------|
| 工人站在原地等机器启动（停住2秒） | ✅ 是等待 | 全身静止超过1秒 |
| 工人伸手过程中的短暂停顿（0.3秒） | ❌ 不算等待 | 时间太短，归入伸手 |
| 工人在思考下一步做什么（停住3秒） | ✅ 是等待 | 全身静止超过1秒 |

**标注步骤**：
```
1. 找到"工人开始停住不动"的时刻 → 记为开始时间
2. 找到"工人开始新动作"的时刻 → 记为结束时间
3. 只有超过1秒的静止才算等待
```

## 三、实际标注操作方法

### 3.1 最简单的标注方法（推荐新手）

**工具**：手机/电脑 + 纸笔

**步骤**：
```
第1步：准备
- 打开视频播放器（可以暂停、慢放）
- 准备纸笔或Excel表格

第2步：看视频
- 第一遍：正常速度看，理解工人做了什么
- 第二遍：慢速（0.5倍）看，逐个动作分析

第3步：记录
每当识别到一个动作，记录：
- 动作类型（R/G/M/RL/W）
- 开始时间（视频上显示的时间）
- 结束时间
- 备注（简单描述）

第4步：填表
整理成表格：
```

**标注表格模板**：

| 序号 | 动作类型 | 开始时间 | 结束时间 | 持续时间 | 备注 |
|------|----------|----------|----------|----------|------|
| 1 | R | 0:00.5 | 0:01.2 | 0.7秒 | 伸手拿螺丝刀 |
| 2 | G | 0:01.2 | 0:01.5 | 0.3秒 | 抓取螺丝刀 |
| 3 | M | 0:01.5 | 0:03.0 | 1.5秒 | 移动到工件处 |
| 4 | RL | 0:03.0 | 0:03.2 | 0.2秒 | 放在工件旁 |
| 5 | W | 0:03.2 | 0:05.5 | 2.3秒 | 等机器启动 |

### 3.2 用Excel标注（推荐）

**创建表格**：

```excel
视频文件名：产线作业视频1.mp4
标注人：XXX
标注时间：2025-05-21

动作标注表：
| 序号 | 动作代码 | 动作名称 | 开始时间(秒) | 结束时间(秒) | 持续时间(秒) | 备注 |
|------|----------|----------|--------------|--------------|--------------|------|
| 1    | R        | 伸手     | 0.5          | 1.2          | 0.7          | 伸手拿螺丝刀 |
| 2    | G        | 抓取     | 1.2          | 1.5          | 0.3          | 抓取螺丝刀 |
| ...  | ...      | ...      | ...          | ...          | ...          | ...  |
```

**Excel公式**（自动计算持续时间）：
```
持续时间 = 结束时间 - 开始时间
在单元格输入：=E2-D2
```

### 3.3 用标注工具（可选）

**免费工具**：
- **ELAN**：专业视频标注工具（功能强大但较复杂）
- **VLC媒体播放器**：简单标注（用书签功能）

**VLC简单标注方法**：
```
1. 打开视频
2. 播放到动作开始点 → 按Ctrl+Shift+B添加书签
3. 播放到动作结束点 → 添加书签
4. 查看→书签列表 → 导出时间点
```

## 四、标注练习示例

### 4.1 练习1：简单装配动作

**场景描述**：
```
工人站在桌前，桌上有螺丝刀。
0.0秒：工人开始动作
0.5秒：手开始向外伸（伸手开始）
1.2秒：手到达螺丝刀位置并停住（伸手结束）
1.5秒：手握住螺丝刀（抓取结束）
3.0秒：手带着螺丝刀移到工件处（移动结束）
3.2秒：手松开螺丝刀（放开结束）
5.5秒：工人开始新动作
```

**正确标注**：

| 序号 | 动作 | 开始 | 结束 | 持续 | 说明 |
|------|------|------|------|------|------|
| 1 | R | 0.5 | 1.2 | 0.7秒 | 伸手 |
| 2 | G | 1.2 | 1.5 | 0.3秒 | 抓取 |
| 3 | M | 1.5 | 3.0 | 1.5秒 | 移动 |
| 4 | RL | 3.0 | 3.2 | 0.2秒 | 放开 |

**注意**：
- 0.0-0.5秒的停顿不算等待（太短，归入准备时间）
- 3.2-5.5秒的停顿算等待（超过1秒）

### 4.2 练习2：识别等待动作

**场景描述**：
```
工人完成装配后：
3.2秒：放下螺丝刀
5.5秒：开始下一个动作（等机器启动）
```

**问题**：3.2-5.5秒应该标注什么？

**答案**：
```
应该标注为Wait（等待）
原因：
- 工人全身静止（没有新动作）
- 持续时间2.3秒 > 1秒（超过最小等待时长）
- 明确是在等机器启动
```

## 五、标注常见问题

### 5.1 问题1：动作边界模糊

**问题**：伸手和抓取的分界在哪里？

**答案**：
```
分界点：手停住 + 开始握

判断方法：
┌─────────────────────────────────────────┐
│ 伸手阶段                                 │
│ - 手在移动                               │
│ - 手臂伸直                               │
│ - 速度较快                               │
│                                         │
│ ↓ 分界点：手停住                         │
│                                         │
│ 抓取阶段                                 │
│ - 手停住                                 │
│ - 手臂开始弯曲                           │
│ - 手握住物体                             │
│ - 速度骤降                               │
└─────────────────────────────────────────┘
```

### 5.2 问题2：短暂停顿算什么？

**问题**：动作中间有0.3秒停顿，算等待吗？

**答案**：
```
不算等待！

判断规则：
- 停顿 < 0.5秒：归入前一个动作的尾部缓冲
- 停顿 0.5-1.0秒：过渡期，看后续动作
- 停顿 > 1.0秒：算等待动作

例子：
伸手→停顿0.3秒→抓取
→ 这0.3秒归入伸手的尾部，不算等待
```

### 5.3 问题3：动作太快看不清

**问题**：抓取动作太快（0.1秒），怎么标注？

**答案**：
```
方法1：慢速播放
- 视频播放器调到0.25倍速
- 每一帧仔细看

方法2：逐帧分析
- 用VLC播放器
- 按E键逐帧前进
- 记录每一帧的动作状态

方法3：保守标注
- 如果实在看不清，标注为"不确定"
- 备注：动作太快，需慢放确认
```

## 六、标注工作量估算

### 6.1 标注一个视频需要多久？

| 视频时长 | 动作数量 | 标注时间 | 说明 |
|----------|----------|----------|------|
| 10秒 | 5-10个动作 | 5-10分钟 | 简单场景 |
| 30秒 | 15-30个动作 | 15-30分钟 | 中等场景 |
| 60秒 | 30-60个动作 | 30-60分钟 | 复杂场景 |

**公式**：标注时间 ≈ 视频时长 × 1-2倍

### 6.2 需要标注多少数据？

**最低需求**（验证规则）：
```
- 2-3个视频
- 每个视频10-30秒
- 总共50-100个动作
- 标注时间：约1小时
```

## 八、标注验证方法

### 8.1 自我检查

标注完成后，检查：
```
✅ 检查1：动作是否连续？
- 每个动作的结束时间 = 下一个动作的开始时间
- 没有遗漏的动作

✅ 检查2：时间是否合理？
- 伸手：通常0.5-2秒
- 抓取：通常0.1-0.5秒
- 移动：通常0.5-3秒
- 放开：通常0.1-0.3秒
- 等待：必须>1秒

✅ 检查3：动作顺序是否合理？
- 正常顺序：R→G→M→RL
- 或者：R→G→RL（没有移动）
```

### 8.2 与系统对比

```
步骤：
1. 你标注 → 人工标注结果
2. 系统分析 → 系统识别结果
3. 对比两者 → 计算准确率

准确率 = 一致的动作数 / 总动作数 × 100%

目标：准确率 > 80%
如果 < 80%，调整识别规则
```

## 九、快速上手步骤

### 9.1 第一次标注（新手）

```
第1步：找一个简单的视频（10-15秒）
第2步：正常速度看一遍，理解工人做了什么
第3步：用纸笔记录每个动作（粗略时间）
第4步：慢速（0.5倍）再看一遍，精确记录时间
第5步：整理成表格
第6步：自我检查（动作连续、时间合理）
```
